How to Import

VirtualBox
  1. Run VirtualBox.
  2. In VirtualBox's File menu, select Import Appliance.
  3. In the Appliance Import Wizard that opens, click choose.
  4. Browse to the VM image file you want to import and click Open.
  5. You can see the VM configurations. Click Import to finish.
VMware Player & Pro
  1. Run VMware Workstation.
  2. Click Open a Virtual machine.
  3. Browse to and open the VM you want to import.
VirtualBox
  1. Double-click downloaded .OVA file.
  2. Click "import".
VMware Fusion
  1. Double-click downloaded .OVA file.
  2. Click "Continue".
  3. Enter where to save virtual machine and click "Save".
  4. Click "Finish".
  5. Click "Upgrade".
VirtualBox
  1. Double-click the downloaded .OVA file.
VMware Player & Pro
  1. Double-click the downloaded .OVA file.
  2. A window opens where you can change the name of the VM and decide where it is saved, or accept the defaults.
  3. Click Import to finish.
You can also import the VM via the Open a virtual machine button while at the VMware home screen.

How to Run

Writing the image to your USB disk

The USB versions of the appliances are provided as Raw Disk Images. They cannot be simply moved to the USB drive, but must be copied byte by byte to make an exact copy of the image in case. Refer to the following instructions to perform the cloning.

For experienced users, the dd tool might be a comfortable option, but it is easier to install and use the application Gnome Disc Tool.
  1. Install the Gnome Disk Tool through the package manager of your Linux distribution. It is usually called gnome-disk-util or gnome-disk-utility.
  2. After installation, run the tool from your application menu or with the command gnome-disks.
  3. Choose your USB drive from the list in the application window.
  4. Delete all existing partitions from the drive by pressing the minus button beneath Volumes.
  5. After removing all of the existing partitions from your USB drive, press the button with three horizontal lines in the top right of the window and choose Restore disk image. A dialog opens where you can browse to and choose the image you want to burn on the drive and start the process.
  6. During the burning of the image, Gnome disk utility displays a progress bar (Figure 3.3). After 100 % is reached, the drive is ready. Press on the same button with three horizontal lines as earlier, and choose Standby now to switch of the drive so it can be removed safely.
  7. The USB appliance is now ready for use.
For burning the image, in addition to using the built-in Disk Utility app you need to download ApplePiBaker 1.9.9 (note that newer versions up from 2.0 are not able to burn the drive properly).
  1. Open the Disk Utility app on your Mac.
  2. Choose the drive you intend to use from the list on the left.
  3. Choose MS9-DOS(FAT) as the format from the drop-down menu in the center of the view and GUID Partition Map as the Scheme below.
  4. Run ApplePiBaker (you need administrator permissions for this).
  5. Choose your USB drive from the pane on the left.
  6. Select your ISO image through the three dots button beside the IMG file field.
  7. Start burning by pressing the Restore Backup button and wait for the burning process to run.
Booting from the USB drive:
  1. Reboot
  2. While startup sound is playing, press down the ALT key.
  3. When a USB device icon appears, select it. System will now boot with the USB disk.
To write the downloaded image into your USB disk on Windows you first need to download USB Image Tool.
  1. Run USB Image Tool.
  2. On the left, select your USB device.
  3. Click the button with a green arrow labeled Restore and browse to the image you want to write on the USB disk. Note that the image is an .iso file - while browsing for the file, select All files (*.*) instead of suggested Image files (*.img; *.ima) to view it.
  4. USBIT ask if you want to restore selected image. Click Yes.
  5. Wait for the green progress bar to finish. After that, your USB device ready for use.

Booting from the USB drive in Windows 10:
  1. From the Start menu, select Power and Restart while pressing down the shift button.
  2. Windows restarts to a blue Boot Options menu.
  3. Click on the the Use a device option and select your USB drive.
