Getting started with virtual machines is straightforward and has very few steps - and you only need to do this once! These instructions should let you accomplish the required steps with ease and then proceed to download your first virtual machine.

What is this virtual machine stuff?

A virtual machine (VM) is an emulation of a computer system. Like a physical computer it can run an operating system and be used just like any other computer - the difference is that the "hardware" it runs on is created by a specific program, a hypervisor. In essence, a virtual machine is a computer inside a computer.

The computer running the hypervisor is usually referred to as host while the virtual machine is called guest.

How does this benefit me? Virtual machines can be created, saved and passed between users. Through this site you can download a virtual machine image that contains pre-installed software and features, saving you the trouble of installing and configuring everything needed on a course separately. Just a couple of steps and you'll be ready to run your very own virtual machine!
Do I need a virtual machine? While a virtual machine can be useful, the use of virtualization is not required on most courses. Check the Moodle site of the course or ask your teacher if you are unsure.
Are there security or privacy risks? Basically, the VM runs on your computer, behind your firewall.

To get started, you first need to check that virtualization is enabled on yor computer.

Enable Virtualization from BIOS Settings

All virtual machines run inside a program called hypervisor, which rely on a virtualization feature in your processor. Unfortunately this is not enabled by default.

To enter BIOS you need to restart your PC and press the appropriate key (usually ESC, DEL, F1, F2 or F10) during the start-up, before your operating system begins to load.

Virtualization is called either VT-x or AMD-V.

Once in your BIOS settings, first take a look at the system information page - some older PCs do not have a configuration for virtualization and simply list it in the first page as "supported". In such case, you don't need to do anything and can move to installing the hypervisor.

For the rest of us, the virtualization setting can often be found under a tab labeled 'Advanced', 'System Configuration' or 'Security'. Once you have spotted a promising title, navigate to the tab in question. While the tab is open, use up and down arrow keys to select the virtualization setting and press the enter key. Choose 'Enabled', press enter and save the changes you made - the instructions are displayed at the top or bottom of your BIOS screen.

Install a Hypervisor

Which OS are you running? The operating system into which the hypervisor is installed into, is called Host OS, and the installation instructions differ slightly between Linux, MacOS and Windows.

Choose your hypervisor between Oracle's VirtualBox or VMWare's VMPlayer.

Download and install the hypervisor. Download links are provided underneath, but for more detailed installation instructions and additional help, head to our Install Hypervisor section.

Install Add-Ons into the Hypervisor

This step is necessary only for VirtualBox! Plain vanilla VirtualBox supports only USB 1.0, which is frankly speaking abysmally slow. In order to support USB 2.0, you need to download extension pack from Oracle's site and install it.

Want to have more detailed instructions? Look no further:

Linux Once you have downloaded the extension pack, run VirtualBox. From the Tools button, open the Preferences menu and choose the Extensions tab. Click on the add icon on the right of the Extension Packages field, browse to and choose the extension file pack you downloaded. Confirm to start the install and follow the hypervisor's instructions.
MacOS Once you have downloaded the extension pack, run VirtualBox. From the Tools button, open the Preferences menu and choose the Extensions tab. Click on the add icon on the right of the Extension Packages field, browse to and choose the extension file pack you downloaded. Confirm to start the install and follow the hypervisor's instructions.
Windows Once the download has been completed, locate the extension pack and double-click on it. Follow the instructions of your hypervisor to complete.

All Done!
