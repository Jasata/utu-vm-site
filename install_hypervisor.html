Obtaining the hypervisor

The virtual machines can be run on most available hypervisors, but support is provided for VMware hypervisors Fusion (macOS) Workstation Pro (Windows, Linux) and Workstation Player (Windows, Linux) as well as Oracle VirtualBox (Windows, Linux, MacOS). The first step to running your virtual machine is to obtain an installer for your hypervisor of choice.

  • VMware Workstation Pro
  • VMware Fusion
  • Vmware Workstation Player

What is the difference and which hypervisor should I choose?

The choice is up to you! Oracle VirtualBox is open-source software, which means that it can be used for free. It is also multi-platform, so it can be used in Windows, Linux and MacOS. VMware offers licenced software, which can be used when a licence is bought or provided e.g. by a university. VMware Workstation Pro can be used in Windows or Linux, while Fusion is for MacOS. VMware also offers a free version, Workstation Player, for Windows and Linux users. It does not have some of the advanced functionalities of Workstation pro, like the ability to run several virtual machines simultaneously, but for running one VM it works fine.

Installing a hypervisor on Windows

VMware Workstation Player & Pro (Windows)

  1. Download the installer file from the links above. Make sure to download the Windows version.
  2. After the download has finished, open the VMware Workstation Pro or Player .exe file.
  3. Follow the installer prompts to complete the installation.

Issues in downloading or installing VMware Workstation? Visit their support site for further instructions.

Oracle VirtualBox (Windows)

  1. Download the installer from the VirtualBOx site.
  2. After the download has finished, double-click on the .exe file to start the installation.
  3. Your system might ask for installation permission several times during the process. Allow it to continue every time.
  4. Follow the installer prompts to complete the installation.

Issues in downloading or installing Oracle VirtualBox? Visit their manual for further instructions.

Installing a hypervisor on macOS

VMware Fusion (macOS)

  1. Download VMware Fusion Pro by clicking the download button on the site linked above. The VMware Fusion .dmg file is saved in your default download directory.
  2. After the download has finished, double click the downloaded file. A window opens.
  3. Double click on the VMware Fusion logo.
  4. Installing requires administrative privileges. Enter your user name and password when prompted.
  5. Wait for the initialization to complete and agree to the terms and conditions at the next prompt
  6. Enter the license key or choose the trial option.

Issues in downloading or installing VMware Fusion? Visit their support site for further instructions.
