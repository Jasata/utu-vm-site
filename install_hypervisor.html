Obtaining the hypervisor

The virtual machines can be run on most available hypervisors, but support is provided for VMware hypervisors Fusion (macOS) Workstation Pro (Windows, Linux) and Workstation Player (Windows, Linux) as well as Oracle VirtualBox (Windows, Linux, MacOS). The first step to running your virtual machine is to obtain an installer for your hypervisor of choice.

  • VMware Workstation Pro
  • VMware Fusion
  • Vmware Workstation Player

What is the difference and which hypervisor should I choose?

The choice is up to you! Oracle VirtualBox is open-source software, which means that it can be used for free. It is also multi-platform, so it can be used in Windows, Linux and MacOS. VMware offers licenced software, which can be used when a licence is bought or provided e.g. by a university. VMware Workstation Pro can be used in Windows or Linux, while Fusion is for MacOS. VMware also offers a free version, Workstation Player, for Windows and Linux users. It does not have some of the advanced functionalities of Workstation pro, like the ability to run several virtual machines simultaneously, but for running one VM it works fine.
